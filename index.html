EDACC

Experiment Design and Administration for Computer Clusters

EDACC is a framework for the design, execution and analysis of large experiments with arbitrary algorithms on computer clusters.

The framework consists of 3 major (core) components:

  • Database: Stores information about algorithm executables, problem instances and experiments
  • GUI application: Java GUI application for managing algorithms and problem instances, creating and analyzing experiments
  • Computation Client: Responsible for the execution of the algorithms on remote, distributed systems such as computer clusters

Optional components:

  • Web Frontend: Web client to perform monitoring and statistical analysis of experiment results
  • Job server: Small application that increases the throughput of the system when large-scale grids are used
  • Visualizer: Visualize the configuration process of algorithms and to analyze the results
  • Automatic algorithm configurator API: Java API for parallel automatic algorithm configurators
  • eAAC: Automatic configurator framework, which includes several automatic algorithm configurators

Source code

EDACC is free, open-source software and available at Github.

Applications
